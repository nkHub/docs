# android

##### 基本项目目录

```
    1. adapter（列表：BaseRecyclerViewAdapter）
    2. net（网络：Retrofit）
    3. bean（数据实体）
    4. module（模块）
    5. intercepter（log拦截器）
    6. database（数据库）
    7. rxbus（通信）
    BaseActivity.java
    BaseApplication.java
    ActivityController.java
```

##### 常用框架

```
    1. 列表：BaseRecyclerViewAdapter
    2. 网络：Retrofit
    3. 图片加载：Glide
    4. 通信：rxbus
    5. 依赖注入：Dagger
```
